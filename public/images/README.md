# LSDB Website Images Directory

This directory contains all visual assets for the LSDB website.

## Directory Structure

```
public/images/
├── hero/              # Hero section images
│   ├── hero-bg.jpg    # Main hero background
│   ├── hero-1.jpg     # Alternative hero image 1
│   └── hero-2.jpg     # Alternative hero image 2
├── institutional/     # Institutional/office images
│   ├── office-1.jpg   # Office exterior
│   ├── office-2.jpg   # Office interior
│   ├── team-1.jpg     # Team meeting
│   └── training-1.jpg # Training session
├── services/          # Service-related images
│   ├── counseling.jpg # Career counseling
│   ├── assessment.jpg # Skills assessment
│   ├── workshop.jpg   # Workshop/training
│   └── support.jpg    # Institutional support
├── artwork/           # Custom illustrations
│   ├── career-path.svg
│   ├── skills-growth.svg
│   └── mentorship.svg
└── logos/             # Brand logos
    ├── lsdb-logo.svg
    ├── lsdb-icon.png
    └── favicon.ico
```

## Image Requirements

### Hero Images
- **Dimensions**: 1920x1080px (16:9 aspect ratio)
- **Format**: JPEG or WebP
- **Quality**: High resolution, professional photography
- **Content**: Career development, learning environments, professional settings

### Institutional Images
- **Dimensions**: 1200x800px (3:2 aspect ratio)
- **Format**: JPEG or WebP
- **Content**: Office spaces, team collaboration, training sessions

### Service Images
- **Dimensions**: 800x600px (4:3 aspect ratio)
- **Format**: JPEG or WebP
- **Content**: Career counseling sessions, skills workshops, professional development

### Artwork
- **Format**: SVG (vector) for scalability
- **Style**: Professional, clean, modern
- **Colors**: Use LSDB brand colors (#0056b3, #2e8540)

## Usage Guidelines

1. **Optimize Images**: Compress images before uploading
2. **Alt Text**: Always provide descriptive alt text for accessibility
3. **Responsive**: Use responsive image techniques (srcset, sizes)
4. **Lazy Loading**: Implement lazy loading for below-the-fold images
5. **Copyright**: Ensure all images have proper licensing

## File Naming Convention

Use descriptive, lowercase names with hyphens:
- `hero-background-career.jpg`
- `service-career-counseling.jpg`
- `institutional-office-exterior.jpg`
