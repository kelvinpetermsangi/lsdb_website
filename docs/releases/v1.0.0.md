# LSDB Website Release v1.0.0

Release date: 2026-03-27

## Summary

`v1.0.0` is the first full production packaging of the LSDB website refresh. It brings the redesigned bilingual experience together with release documentation, SMTP-backed contact delivery, and a VPS deployment pack for Namecheap hosting.

## What's Included

- modern homepage distinct from the rest of the site
- refreshed About, Services, Programs, Resources, Team, and Contact pages
- centralized language and content management for English and Kiswahili
- updated LSDB logo, favicon, and downloadable business profile
- footer simplification with legal links and profile download only
- WhatsApp quick-help CTA using the official LSDB Tanzania number
- canonical URLs, sitemap, and robots support
- PM2, Nginx, and standalone build configuration for VPS hosting
- SMTP-ready contact delivery through Nodemailer

## Operational Notes

- Production contact delivery depends on the SMTP variables listed in `.env.example`.
- The app is configured for reverse-proxy deployment behind Nginx on `127.0.0.1:3000`.
- The business profile file shipped in this release is `public/documents/lsdb-business-profile.pdf`.

## Validation

- `npm run build`
- live verification of `/`, `/contact`, `/organization-profile`, and the PDF asset
